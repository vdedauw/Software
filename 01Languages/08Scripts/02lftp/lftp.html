1 Sun Jun 9 12:30:16 2019 lftp.sh 01ltp Publish or backup a collection of files to/from an ftp server
    commands
  • publish, publish files and directory structure from localdir to remotedir
  • backup, backup files and directory structure from remotedir to (new) localdir
  • show, do nothing, show parameter values
  • checks, do nothing, show parameter checking order
  • usage, shows the script usage
$1   SILENCE optional 'silence' or 'SILENCE', when set the script will not do 'echo'
$2   TAG TAG, the tag used for the 'logger', /var/log/syslog entry
$3   HOST host=, the host with the ftp service
$4   PORT port=, de remote port, default: 21
$5   NOPORT noport=true, no port checking (nmap) will be performed, default: false
$6   LOCAL_DIR localdir=, the local directory containing the files
$7   REMOTE_DIR remotedir=, the remote directory on the ftp server
$8   USER user=, the user used to login into the ftp server
$9   PASSWORD password=, the password used to login into the ftp server
can be ommited when the user is 'anonymous'
