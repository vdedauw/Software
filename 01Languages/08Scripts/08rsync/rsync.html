1 Mon May 6 00:55:25 2019 rsync.sh sync directories, using rsync
  rsync.sh source=path target=path delete=false
 
  • when source path is closed with a '/' like '/data/', data will not be created
  • when source path is not closed with a '/' like '/data', data will be created before the sync process starts
  • the delete argument is optional, when not given --delete is used
