isexecutable

1 Logic

  • read FIRST command line parameter: 'SILENCE', if found, get rid of it
  • read FIRST command line parameter: 'TAG', get rid of it
  • read FIRST and all further parameters as executable file names
  • checks if a 'command line parameter' exists as a file.
  • if the file exist, check for executable, -x
  • return 0, ok, all given files exist and are executable
  • return 1, not ok, no command line parameters
  • return 2, not ok, no executables, only two parameters given, first is SILENCE
  • return 3, not ok, file does not exist
  • return 4, not ok, file exist, but not executable
