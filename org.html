2 Environment

The environment is strongly dependent on the language the project is written in.
In the most simple case, in the shell bash language, a single computer with an OS, and a bash shell is sufficient.
In more complex cases, in PHP, a webserver with PHP enabled is needed, also a client computer with a web browser.
If the tests are written for a microcontroller program, the corresponding hardware will be needed, and software to program the controller need to be present.

In most projects the programming environment is not the same as the run environment.
